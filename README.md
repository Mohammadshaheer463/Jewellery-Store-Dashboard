# Jewellery POS & Inventory Management System (Case Study)

A full-featured desktop POS and inventory management system built for a jewellery client as a solo developer.

Full source code is private due to client confidentiality.

---

## Features

- Inventory Management (Gold, Stones, Jewellery)
- Real-time Price Estimation Engine
- Product Barcode & Tag Generator
- POS Billing System (Multi-item Sales)
- Customer Management & Purchase History
- Admin & Salesman Role Control
- Discount Engine
- Raid Mode for Data Privacy
- Invoice Export to Excel
- Image Compression & Storage
- USB Barcode Scanner Support

---

## Tech Stack

- Python
- PySide6 (GUI)
- SQLite
- PIL (Image Processing)
- Pandas (Invoice Export)
- Threading & Async Workers

---

## System Architecture

GUI → Business Logic → SQLite Database  
Barcode Generator → PIL Image System  
POS → Sale Items → Customer Mapping → Excel Export
